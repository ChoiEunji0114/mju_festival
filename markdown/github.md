Github 명령어 모음
------------------

#### Pull(깃허브 -> 로컬, 되도록이면 자주)

```
git pull
```

#### Stage(작업한 모든 파일 업로드 시 .(온점) 표기)

```
git add 파일명
```

#### Stage 취소

```
git reset
```

#### Commit(메세지는 작업한 내용이 잘 드러나게 작성)

```
git commit -m "커밋 메세지"
```

#### Push(로컬 -> 깃허브)

```
git push origin 브랜치명
```

#### 브랜치 이동(ex. 브랜치0 -> 브랜치1)

```
git checkout 브랜치1
```
